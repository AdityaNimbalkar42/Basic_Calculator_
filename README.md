**BASIC CALCULATOR**

This is a simple Calculator made using HTML, CSS, and JavaScript, it performs basic arithmetic operations.The calculator can add, subtract, multiply, and divide numbers.
It also has a clear button to reset the calculation.

**Installation**

*  Download or copy all the project files into one folder.

*  Make sure the files are named correctly.

*  No extra software or libraries are needed.

**Run** 

*  Open the folder.

*  Double-click on index.html.

*  The calculator will open in your web browser.

**Usage**

*  Click the number buttons to enter values.

*  Click an operator (+, −, ×, ÷) to choose an operation.

*  Click the equals (=) button to get the result.

*  Click the C button to clear everything.
